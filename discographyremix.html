Remix

最近のボクは様々な音ゲー曲のRemixを作っています。その中でも特に気に入っているものを紹介します。

VERIAL -the spectrum-

BPM:190 　GENRE:Hardcore Techno

いきものがかりさんの「熱情のスペクトラム」のRemixで、ボクが初めて作ったRemixです。 先程ボクは「音ゲー曲のRemixを作っています」と書きましたが、これは例外で、J-POPのRemixです。 初めてのRemixでJ-POPを音ゲー曲っぽくインストアレンジしようという無茶をしていました。 このRemixの特徴としてあるのが、最初は原曲キーより+1された音程で、ラスサビで原曲キーに戻るという少し特殊な進行です。 Remixに色を出そうとした結果です。知識がないなりに。 また、尺稼ぎとしてBメロとラスサビの合間にダブステップ地帯が入っています。 当時はそういうのにハマっていました。 あまりにも探り探りのRemix、お楽しみください。

RIKU / Gori-Tama · VERIAL -the spectrum-

FREEDOM DiVE↓ （RIKU Remix）

BPM:222 　GENRE:Happy Hardcore

「BMS」より、xiさんの「FREEDOM DiVE↓」のRemixです。 音ゲー曲の中でも随一の知名度と人気を誇る曲とあって、プレッシャーを勝手に感じながらRemixしました。 原曲は音がキラキラした壮大な楽曲ですが、ボクのRemixは結構サッパリしています。 まあ、当時壮大な音を作る方法を知らなかったのもありますが...... この曲は、Twitterで公開した時やたらと反響が大きかったのも思い出深いです。 ......今となってはからっきしですが。

RIKU / Gori-Tama · FREEDOM DiVE↓ （RIKU Remix）

Viyella's Tears -Like Against Remix-

BPM:218 　GENRE:Classical Hardcore

「オンゲキ」より、Laurさんの「Viyella's Tears」のRemixです。 いつものRIKUではなく、Like（ライク）名義のRemixになっています。 Like名義の時は「Speedcoreやろーぜ」って感じのスタンスです。 というわけで、サビのところで「Ready Go!」のボイスを合図にSpeedcoreになります。 Symphonicなブレイクとのギャップをお楽しみください。 それと、このRemix、SoundCloudではやたらと評判が良いのです。 リスナーの皆さま、ありがとうございます。今後ともよしなに。

RIKU / Gori-Tama · Viyella's Tears -Like Against Remix-

Hartmman Dream

BPM:180 　GENRE:Hardcore Techno

「東方地霊殿」より、「ハルトマンの妖怪少女」のRemix......というよりアレンジです。 ボクの初の東方アレンジ曲です。 元々7拍子と4拍子が交互にやってくるという非常に特徴的な曲で、ビートを刻むのが少し難しかったりしました。 でも、元のメロディーが抜群にいいので、雰囲気は出し易かったです。 少しアレな話ですが、尺を稼ぐ方法にも苦心しまして、最終的にイントロとアウトロにシンフォニックな部分を作りました。 こんな感じで苦労して完成させた時には、東方アレンジって難しいけど楽しいなーと本当に思いました。 ちなみに、曲名の由来ですが、東方M-1のとあるネタです。 あと、古明地こいしのことは大好きです。

RIKU / Gori-Tama · Hartmman Dream

Grievous Lady -Unidentfied Upon-

BPM:210 　GENRE:Classical Hardcore

「Arcaea」より、Team Grimoireさん vs Laurさんの「Grievous Lady」のRemixです。 RIKU vs Satinとして一人合作した初のRemixでもあります。 Grievous Ladyといえば、Arcaeaの永遠のボス曲として有名ですよね。 この曲をRemixするにあたってそれ相応にハードルが高かったので、マジの本気でやらせて頂きました。 制作中はとにかく「原曲に負けない」ということを意識していました。 原曲は2人、こちらは1人。めっちゃ大変でしたが、オリジナルとの差を出しつついい感じにできたと思います。 SoundCloudにおいても結構再生数が多いので、頑張った甲斐がありました。嬉しい。

RIKU / Gori-Tama · Grievous Lady -Unidentfied Upon-

Fracture Ray -Vacuous Memory-

BPM:200 　GENRE:Symphonic Dubstep

「Arcaea」より、削除さんの「Fracture Ray」のRemixです。 合作相手のAKATUKIさんについてですが、この人の中身はボクじゃないです。マジの友人です。 なので、これは正真正銘初の合作Remixです。 Fracture Rayは、上記のGrievous Ladyと同じくArcaeaの大ボス曲として有名です。やっぱりハードル高い。 また原曲に負けないように頑張りたかったのですが、初の合作ということもあってAKATUKIさんと上手く連携が取れませんでした。 今だからこそリベンジしたい曲です。 Grievous Ladyともども、そのうちリメイクするかも。

RIKU / Gori-Tama · Fracture Ray -Vacuous Memory-

YURUSHITE（WTF Remix）

BPM:270 　GENRE:Speedcore

「オンゲキ」より、t+pazoliteさんの「YURUSHITE」のRemixです。 RIKU vs Likeの一人合作Remixです。 原曲の時点でさえ、その超高BPMの影響でとっても激しい曲なのですが、ボクはそれに飽き足らず「更なる激しさ」を求めました。 その結果、オリジナルにLike成分を加え、すんごい激しいRemixが誕生しました。 原曲よりも重く、うるさく、数が多いキックを大音量で浴びれば、あなたの鼓膜が漏れ無く破壊されるでしょう。 ちなみにこの曲、Twitterで公開したら1000回再生を突破しました。なんでだろう？ あと、「WTF」の意味については......まあ調べなくてもいいと思います。

RIKU / Gori-Tama · YURUSHITE（WTF Remix）

星屑ストラック -Trinity-

BPM:220 　GENRE:Hardcore Techno

「太鼓の達人」より、t+pazoliteさんの「星屑ストラック」のRemixです。 Vivi Preferinde（ヴィヴィ・プリフェリンデ）名義の初登場です。 Vivi Preferindeは、RIKU,Satin,Likeが集合した名義で、それぞれの名義の特徴が集合した楽曲の際に使用されます。 だから"Trinity(三位一体)"なわけですね。 バイオリン、ワブルベース、ガバキック、荒ぶるハイハットと、色々な要素をぶち込まさせていただきました。 終盤では、エモエモな展開をもっとエモくするため、一工夫しています。 これを聴いて、みなさんも星屑ストライカーになりましょう。

RIKU / Gori-Tama · 星屑ストラック -Trinity-

Coquette：Neo Style

BPM:200 　GENRE:Neo Artcore

「太鼓の達人」より、Jun Kurodaさんの「Coquette」のRemixです。 Quina Ria（クイナ・リア）名義の初登場曲です。なんとなく女性っぽい名前にしてみました。 この名義の時は、他の3つの名義とは「一味違う」感じのことをしようと思っています。 このRemixの一味違うポイントは、「何でもしちゃってる」というところです。 今までやったことのなかったような音の組み合わせをしたりだとか、特殊なリズムでビートを刻んだりとかです。 このRemix以降、なんとなく自分の音のレパートリーが増えたように感じたりもします。 ちなみにジャンルですが、ただのArtcoreじゃありません。Neo Artcoreです。 意味は特にないです。

RIKU / Gori-Tama · Coquette：Neo Style

Radiance -Never Last-

BPM:170 　GENRE:Trancecore

「オンゲキ」より、Nhatoさんの「Radiance」のRemixです。 このRemixは、ジャケットに描かれたオンゲキのキャラクター「珠洲島有栖」の誕生日を祝おう、という感じで作りました。 Twitterで公開するために2分20秒までに収めたかったので、原曲と比べて大分BPMが高くなっています。 Radianceはオンゲキの曲の中で特に大好きで、このRemixはとにかく「すき！！」という想いを込めました。 サビ辺りでヘッドフォンの左右を飛び回っている音があるとか、最後の方の激しいビートはオンゲキの譜面と対応しているとか、工夫したところは色々あります。 が、この曲に関しては聞くより聴いて貰った方が早いと思うので、とりあえず聴いてくだしあ！ じゃじゃーん！

RIKU / Gori-Tama · Radiance -Never Last-

HONEY-Q（Renaissance Tune Remix）

BPM:190 　GENRE:Renaissance Tune

「オンゲキ」より、こふさんの「HONEY-Q」のRemixです。 上述したRadianceのRemixと同じような感じで、オンゲキのキャラクター「東雲つむぎ」の誕生日記念の曲です。 「Renaissance Tune」とかいう得体の知れないジャンルが書いてありますが、ボクのオリジナルのジャンルです。 ChiptuneのサウンドでHard Renaissanceっぽいメロディを鳴らす、そんな感じです。 それと、この曲を期に、DAWが今までのGaragebandからLogic Pro Xに変わり、「Sylenth1」などの有料プラグインを取り入れました。 流石は有料DAWと有料プラグイン、曲のクオリティが一気に進化しました！ このRemixでボクの成長を少しでも感じていただけたら嬉しいです。

RIKU / Gori-Tama · HONEY-Q（Renaissance Tune Remix）
